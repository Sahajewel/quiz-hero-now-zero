Welcome to Quiz Hero

1. How do you select an element by its class in JavaScript?

Answer: Use `document.querySelector('.className')` to select the first element with the given class.

2. How do you add an event listener to an element?

Answer: Use `element.addEventListener('event', callbackFunction)` to attach an event listener.

3. How do you dynamically change the text content of an HTML element?

Answer: Use `element.textContent = 'New Text';` to modify the text content of an element.

4. How do you toggle a class on an HTML element?

Answer: Use `element.classList.toggle('className')` to add the class if it's missing or remove it if it exists.

5. How do you create an HTML element using JavaScript?

Answer: Use `document.createElement('tagName')` to create a new element and `appendChild` or `append` to add it to the DOM.
